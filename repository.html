Source Repository

Modules

ProxyToys project consists of several modules:
  1. proxytoys - the core module
  2. website - the module contains the documentation and assembles the src and bin distributions
  3. example-code - the example code used in the Javadoc of the package descriptions
When you clone the Git repository you'll get all the modules.
